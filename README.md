# Weather Sounds App

Замечательное приложение позволит наслаждаться звуками природы не отходя от рабочего места. Выбирайте любимую погоду - солнечный летний день, весенний ливень с раскатами грома или зимний снегопад, настраивайте громкость.

Нажатие на кнопку - начало звука, повторное нажатие - пауза. Ползунок под кнопками позволяет настроить громкость.

## Локальный запуск приложения

Для запуска установите зависимости

```
npm install
```

соберите проект

```
npm run build
```

перейдите в папку dist

```
cd dist
```

запустите проект локально

```
npx serve
```

## Запуск в режиме разработки

Дорабатывайте проект по своему вкусу. Установите зависимости

```
npm install
```

запустите проект в режиме разработки

```
npm run start
```
